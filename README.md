# pvzHE-Archiver
 植物大战僵尸杂交版存档管理工具

![Screenshot 2024-06-01 162632](https://github.com/HNRobert/pvzHE-Archiver/assets/120773486/aed24a59-5555-4c50-a25c-d12f9d5dcd01)

## 功能概述

本软件适用于植物大战僵尸杂交版的【游戏存档管理】，

可以在退出游戏返回主菜单时保存游戏进度，

也可以自动在无尽模式进入下一个关卡时自动保存，
并在历史任意关卡的任何时间节点重新切入。

还有备注功能便于管理归档。

## Postscript

当前软件处于测试阶段，
仅支持抽奖盒子的关卡名称显示。
后续将继续更新
