# pvzHE-Archiver
 植物大战僵尸杂交版存档管理工具

![image](https://github.com/user-attachments/assets/c24d0cd4-e72e-4374-8ef6-2cde306a5fb2)

## 功能概述

本软件适用于植物大战僵尸杂交版的【游戏自动存档+存档管理】，

可以在退出游戏返回主菜单时保存游戏进度，

也可以自动在无尽模式进入下一个关卡时自动保存，
并在历史任意关卡的任何时间节点重新切入。
可以有效减轻游戏闪退给你带来的惨重的损失

还有备注、筛选、排序等功能便于管理归档。


### v1.x 大版本完整功能支持列表：

1. 在退出游戏返回主菜单时保存游戏进度。
2. 自动在无尽模式进入下一个关卡时自动保存。
3. 在历史任意关卡的任何时间节点重新切入。
4. 便于系统化管理归档的备注功能。
5. 存档筛选与排序功能。
6. 批量导入与导出存档功能。
7. 自动删除保存时间过长存档的功能。
